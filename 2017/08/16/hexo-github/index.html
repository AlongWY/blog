开启 Travis-CI

使用Github账号登录Travis-CI网站，开启你的仓库。 而后你可以在左侧的My Repositories侧边栏下看到你的仓库，打开它。 在右侧的More options下选择Settings选项，打开General选项卡下的三个选项。 * Build only if .travis.yml is present * Build branch updates * Build pull request updates

然后到Github的Setting页面中完成Access Token的申请，只需要勾选repo这一项权限即可，然后将其作为环境变量GH_TOKEN的值存在Environment Variables栏下。
